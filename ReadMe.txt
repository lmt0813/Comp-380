Commands to Commit and Push
 - git add *filename*
 - git commit -m 'message'
 - git push

Command to Pull
-Make sure to pull for any updates we make to anything
 - git pull

Command to check status of files
- git status
  - checks what files have changed on local repository compared to remote repository

Command to check commits
- git log
  - prints log of commits on screen
  - arrow keys to scroll
  - press q to quit
